## Requerimientos

- NodeJs
- NPM

## Despliegue

- Clonar el proyecto
- Ejecutar `npm install` para instalar dependencias
- Ejecutar `npm start` para lanzar el proyecto
- En en navegador, ir a la ruta http://localhost:3000
